VaultPlane Gateway is installed.

1. Port-forward the proxy port:

     kubectl --namespace {{ .Release.Namespace }} port-forward \
       svc/{{ include "vaultplane.fullname" . }} \
       {{ .Values.service.proxyPort }}:proxy

2. In another shell, call the proxy like OpenAI:

     curl http://localhost:{{ .Values.service.proxyPort }}/v1/chat/completions \
       -H "Content-Type: application/json" \
       -d '{"model":"smart","messages":[{"role":"user","content":"hi"}]}'

3. Port-forward the admin port:

     kubectl --namespace {{ .Release.Namespace }} port-forward \
       svc/{{ include "vaultplane.fullname" . }} \
       {{ .Values.service.adminPort }}:admin

   Then:

     curl http://localhost:{{ .Values.service.adminPort }}/admin/status
